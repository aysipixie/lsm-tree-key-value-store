LSM Tree Architecture
Data Flow in LSM Key-Value Store
1. Write Operations
Operations logged to WAL first, then written to Memtable
2. Memtable
In-memory sorted structure (max 30 entries)
3. SSTables
Persistent sorted disk files
4. Compaction
Merges SSTables to optimize storage and performance
5. Read Operations
Search Memtable first, then SSTables (newest to oldest)
